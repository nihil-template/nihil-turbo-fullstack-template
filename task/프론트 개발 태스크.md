# `apps/web` UI 라우트 구조 (Next.js App Router) - 투두리스트

`apps/api`의 API 엔드포인트를 기반으로 `apps/web`의 UI 라우트 구조를 Next.js App Router 형식에 맞게 설계합니다.

## 현재 문제점 및 이슈

### 🔴 긴급 해결 필요

#### 1. Zod 호환성 문제 ⚠️

**문제**: `nestjs-zod`와 `@nest-zod/z` 패키지가 Zod v4와 호환되지 않을 수 있음

- **현상**: 프로필 수정 시 `zod.defaultErrorMap is not a function` 에러 발생 가능성
- **영향**: 프로필 수정 페이지 API 호출 시 500 에러 발생 가능성
- **해결 방안**:
  1. nestjs-zod 제거 후 직접 Zod 검증 구현
  2. 또는 Zod v3로 다운그레이드 (임시 해결책)
